## Step 3 — Snake Animation

Create `.github/workflows/snake.yml` in your profile repo:

```yaml
name: Snake Contribution Graph

on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:
  push:
    branches: [main]

jobs:
  snake:
    permissions:
      contents: write
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk/svg-only@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark

      - uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

Commit, push, then go to **Actions** and trigger it manually once. It will refresh every 12 hours after that.

---

## Step 4 — Auto Blog Posts

Create `.github/workflows/blog-posts.yml`:

```yaml
name: Latest blog post workflow

on:

    - cron: '0 */6 * * *'
  workflow_dispatch:

jobs:
  update-readme:
    runs-on: ubuntu-latest
    steps:

      - uses: gautamkrishnar/blog-post-workflow@v1

```
